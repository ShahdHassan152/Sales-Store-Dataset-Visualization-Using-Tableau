# Sales-Store-Dataset-Visualization-Using-Tableau


Overview


This project delivers an interactive Sales Performance Dashboard built using Tableau. It transforms raw sales data from Excel into meaningful business insights that help stakeholders track performance, identify trends, and support data-driven decisions.



Data Source
Sales.xlsx

Order details

Customer information

Product categories

Pricing and discounts

Geographic data



Objectives
Analyze overall sales performance.

Identify top products, customers, and regions.

Evaluate the impact of discounts on revenue.

Visualize trends over time.



Key Features
Calculated total sales after discounts.

Time-series analysis based on order dates.

Breakdown of sales by:

Product category

Customer type

Country and city

Interactive Tableau dashboard for executive-level reporting.



Tools & Technologies
Tableau

Microsoft Excel



Outcome


The dashboard provides a clear overview of sales drivers and performance metrics, enabling faster and more informed business decisions.



Skills Demonstrated


Data Visualization • Business Analysis • Tableau Dashboards • KPI Design • Data Preparation
